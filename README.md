Программа моделирования работы волнового твердотельного гироскопа.
На основании упрощенных математических моделей симулировались выходные данные ВТГ 
(угол поворота основания, угол отставания волновой картины) при различных входных данных (угловых скоростях).

- В программе используется две библиотеки numpy; matplotlib.

Установка

- Скопируйте репозиторий
Откройте терминал и выполните:
```bash
git clone https://github.com/Timofey-Maz/modeling_vtg.git
cd modeling_vtg
```
- Установите необходимые библиотеки;
- Запустите основной скрипт:
``` bash
python PZ_2.py
```
